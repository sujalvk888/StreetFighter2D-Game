# ⚔️ Street Fighter 2D

Welcome to the official source code repository for **Street Fighter 2D**, a high-fidelity 2D fighting game developed entirely in Unity WebGL. Experience high-impact combat mechanics, fluid 60 FPS sprite animations, and an adaptive AI system.

## 🕹️ Game Features
* **Arcade Mode:** Battle against a calculating, responsive AI CPU that blocks, adapts, and punishes mistakes.
* **Versus Mode:** Classic 1v1 local multiplayer on a shared keyboard to settle the score with friends.
* **Practice Mode:** Master your character timings, test combos, and explore hitboxes against an infinite-health training dummy.
